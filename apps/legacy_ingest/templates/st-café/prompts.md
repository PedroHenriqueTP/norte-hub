# Prompt Template - Landing Page Premium para Negócios

## 📋 Template Base

Use este template para criar landing pages modernas e aconchegantes para diferentes tipos de negócios. Basta substituir as seções marcadas com `[VARIÁVEL]` pelos dados específicos do seu negócio.

## 📝 Checklist de Personalização

Ao usar o template, certifique-se de definir:

- [ ] **Tipo de Negócio**: Cafeteria, hamburgueria, bar, restaurante, etc.
- [ ] **Paleta de Cores**: Fundo, header e acento
- [ ] **Slogan Principal**: Frase de impacto para o hero
- [ ] **3 Imagens Hero**: Descrição das imagens do carrossel
- [ ] **História do Negócio**: Texto "Quem Somos"
- [ ] **4 Produtos/Serviços**: Principais ofertas
- [ ] **Lead Magnet**: Oferta de desconto/brinde
- [ ] **Redes Sociais**: Links do Instagram, Facebook, WhatsApp
- [ ] **Endereço e Horários**: Informações de localização

---

## 🚀 Como Usar

1. **Escolha o tipo de negócio** ou use o template base
2. **Substitua todas as variáveis** [ENTRE_COLCHETES] com as informações específicas
3. **Cole o prompt completo** no chat com a IA
4. **Aguarde a criação** dos arquivos HTML, CSS e JavaScript
5. **Personalize** conforme necessário

---

## 💡 Dicas Extras

- **Imagens**: Quanto mais específica a descrição, melhor a qualidade das imagens geradas
- **Cores**: Use ferramentas como [Coolors.co](https://coolors.co) para criar paletas harmoniosas
- **Slogans**: Mantenha curtos (máximo 8 palavras) e impactantes
- **Lead Magnet**: Ofertas entre 10-20% funcionam melhor para conversão
- **Mobile First**: Todos os templates são responsivos por padrão
