# DS: Integração de Data Science com Norte Systems

## 🔗 A Ponte Acadêmica

Esta nota conecta os estudos de Data Science da PUC-SP com as verticais operacionais do ecossistema Norte.

## 📦 Ativo Acadêmico Integrado
- **`dashboard_analise_app.py`**: Script Python de análise de dados (pandas, streamlit, numpy, openpyxl). Serve como protótipo para painéis de dados do Norte Systems.

## 🚚 Norte Logistics: Predição de Demanda
Modelos supervisionados de regressão podem ser aplicados para:
- **Predição de Rotas** (Norte Move): Estimar tempo de chegada com base em histórico de viagens (`MoveTrip`).
- **Otimização de Carga** (Norte Logistics): Previsão de volume de entregas por região de SP usando séries temporais.

## 🏥 Norte Systems: Inteligência Clínica
- **Norte Clinic**: Modelos de clustering para identificar perfis de pacientes com maior frequência de consultas.
- **Norte Fit**: Análise de progressão de cargas e macros para recomendar ajustes automáticos no treino.
