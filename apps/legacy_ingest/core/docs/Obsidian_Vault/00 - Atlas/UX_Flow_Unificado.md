# UX Flow: Navegação Unificada Norte Systems

## Fluxo de Acesso
1. **Login Único (Norte Hub)**: Autenticação central.
2. **Vault Selector**: Menu lateral com acesso a todas as verticais.
3. **Dashboard Contextual**: Cada app (Fit, Clinic, Agency) mantém a mesma linguagem visual (Light Mode).

## Comunicação (DM Unificada)
- Um sistema de chat persistente que acompanha o usuário.
- Permite falar com o Suporte Norte ou profissionais (Médicos/Personais) sem trocar de aplicação.
