# Design System Norte

## Identidade Visual (Norte Clean)
- **Modo**: Light Mode
- **Cores Primárias**: Emerald-600 (#059669), Orange-500 (#f97316)
- **Cores de Fundo**: Branco (#FFFFFF), Soft Gray (#F9FAFB)
- **Estética**: Minimalista, Moderno, Premium.

## Modelo de Negócio
- **Arquitetura**: Multi-tenant PaaS.
- **Integração**: Ecossistema unificado de serviços locais.
- **Diferencial**: Perfil Profissional Único (Motorista/Entregador/Profissional) e Fintech integrada.
