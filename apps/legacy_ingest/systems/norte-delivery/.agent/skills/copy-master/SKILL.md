
# Skill: Copy Master (Conversion Writing)

## Descrição
Esta skill é responsável por criar todos os textos (H1, H2, CTA, Body) do site. O foco NÃO é ser poético, é ser PERSUASIVO e CLARO.

## Frameworks Obrigatórios
Ao gerar texto para seções específicas, aplique estes modelos mentais:

### 1. Hero Section -> [A PROMESSA]
- **Estrutura:** Gancho (Headline) + Como funciona (Subhead) + Ação (CTA).
- **Regra:** Fale do benefício final, não da ferramenta.
- *Exemplo Ruim:* "Melhor sistema de delivery."
- *Exemplo Bom:* "Entregue mais rápido e pare de perder dinheiro com taxas abusivas."

### 2. Features -> [BENEFÍCIO > RECURSO]
- Nunca liste apenas a funcionalidade. Diga o que ela resolve.
- **Fórmula:** "Recurso X" que permite "Benefício Y" para que você "Evite Dor Z".

### 3. Seção de Problemas -> [FRAMEWORK P.A.S.]
- **P (Problem):** Identifique a dor (Ex: Comida fria, taxas altas).
- **A (Agitation):** Pise na dor (Ex: Clientes reclamando e margem de lucro zerada).
- **S (Solution):** Apresente o nosso SaaS como o alívio imediato.

## Diretrizes de Tom de Voz (Delivery Platform)
- **Vibe:** Enérgico, apetitoso, rápido, parceiro do dono do restaurante.
- **Palavras-chave:** Velocidade, Lucro, Controle, Sabor, Quentinho.
- **Proibido:** "Soluções integradas de logística gastronômica" (Muito corporativo). Use "Seu delivery voando".

## Ação de Edição
Se o usuário pedir "Melhore esse texto", analise:
1. Está claro?
2. Tem voz passiva? (Mude para ativa).
3. O CTA é fraco? (Mude "Enviar" para "Quero Vender Mais").
