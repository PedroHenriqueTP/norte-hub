
# Skill: UI Artist (Image Gen Prompts)

## Descrição
Gera prompts otimizados para criar assets visuais (ícones, ilustrações, fotos) que mantenham a consistência estética do "FastFeast".

## Estilo Visual (Brand Guide)
- **Estilo:** 3D Render Minimalista (Estilo 'Claymorphism' ou 'Glassmorphism'), Fundo limpo, Iluminação de estúdio suave.
- **Cores:** Laranja Queimado (Brand), Branco, Cinza Suave.
- **Vibe:** High-end tech meets delicious food.

## Templates de Prompt

### 1. Para Ícones/Bento Grids (3D Icons)
`[OBJECT] icon, 3d render, clay style, cute, soft lighting, orange and white color palette, high fidelity, 4k, isometric view, transparent background --v 6.0`

### 2. Para Imagens Hero/Background
`abstract background representing [CONCEPT: speed/delivery], soft gradient, orange and cream colors, glassmorphism texture, noise grain, blurred details, wallpaper style, high resolution --ar 16:9`

### 3. Para Ilustrações de Comida
`delicious [FOOD ITEM], hyper-realistic 3d render, flying in the air, dynamic angle, studio lighting, depth of field, appetizing, condensation droplets, vibrant colors --style raw`

## Como Usar
Quando o usuário pedir "Crie uma imagem para a seção X":
1. Entenda o contexto da seção.
2. Gere o PROMPT exato para ser colado no gerador de imagem (DALL-E/Midjourney).
3. Sugira onde essa imagem deve ser posicionada no layout CSS.
