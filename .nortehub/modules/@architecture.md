# Arquitetura: Norte Lab

**Stack Principal (SaaS Multi-Tenant de Alta Performance)**:
- **Framework Frontend/BFF**: Next.js (App Router)
- **Backend/API**: NestJS (Fastify)
- **Banco de Dados**: PostgreSQL
- **ORM**: Prisma ORM
- **Comunicação Real-Time**: SSE (Server-Sent Events)

**Diretrizes**:
- Tipagem estrita com TypeScript.
- Código limpo sem comentários extensos ou redundantes no meio das funções.
- Prioridade em legibilidade e arquitetura limpa (Clean Architecture / DDD adaptado).
