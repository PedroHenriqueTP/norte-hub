# Segurança e Multi-tenant: Plataforma de Gestão de Projetos e Criação de Produtos (Norte Lab)

**Isolamento Multi-tenant**:
- Isolamento de dados a nível de banco de dados para garantir segurança estrita entre inquilinos.

**Criptografia**:
- **AES-256-GCM** utilizado para proteger dados em repouso e, especificamente, para o OAuth Vault.
